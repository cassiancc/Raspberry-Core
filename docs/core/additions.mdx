## Blocks
- Ash Ovens, crafted from Ash Bricks (Supplementaries).
- Silt Ovens, crafted from Silt Bricks (Twigs).
- Lead Grate, crafted from Lead Blocks (Oreganized)
- Deepslate Gravel, which can be raked.
- Blackstone Gravel, which can be raked.
- Pink Petals, backported from Minecraft 1.20.
- Wildflowers, backported from Minecraft 1.21.5.
- Clovers, a new natural block textured and modelled by ProbablyEkho.
- Ash Blocks, crafted from Ash (Supplementaries).
- Brick counters, as well as Silt and Ash brick variations, a new type of block for decorating your kitchen (@kobber)
- Wildflower Garlands, placeable on walls or worn as flower crowns, crafted out of wildflowers (textures and implementation by Kobber, based on existing textures by ProbablyEkho)
- Storage block for Ash (Supplementaries).

## Items
- Rose Gold Bombs, which deal knockback but not damage. (v1.8+)
- Swap Arrows, which swap the position of an entity with the target.


## Features
- Copper Armour (Copperized) will grant the new Aftershock effect when struck by lightning. This effect increases attack speed and movement speed.
- Copper Armour (Copperized) will grant Lightning Resistance (CoFH Core) effect.
- A port of [Audaki Cart Engine](https://modrinth.com/mod/audaki-cart-engine) to Forge - Minecart speed has now been buffed to 34 meters per second. This should not affect Minecart Contraptions as this buff only applies to carts with players in them.

## Tags
- Items in the `raspberry:enchantable/weapon` tag can use weapon enchantments.
- Items in the `raspberry:enchantable/heartfelt` tag can be enchanted with Heartfelt. (Mining Master)
